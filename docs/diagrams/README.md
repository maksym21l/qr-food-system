# diagrams/

Сюди складай усі діаграми та макети.

Рекомендована структура:
- bpmn/ — BPMN процеси
- uml/ — Use Case Diagram, Activity Diagram, інші UML
- wireframes/ — макети (Balsamiq / Figma експорти)
- erd/ — ERD/схема БД

Що краще зберігати:
✅ PNG/JPG — швидко дивитись в GitHub
✅ SVG — ідеально для схем, не псується якість
✅ PDF — якщо діаграма велика/на кілька сторінок
✅ .bpmn / .xml — вихідник з bpmn.io (краще додавати разом з PNG/SVG)
✅ .bmpr / .balsamiq / посилання на Figma — вихідники макетів (опційно)

Правило:
- Для кожної діаграми: зберігай **і картинку (PNG/SVG)**, і **вихідний файл**, якщо він є.
- Назви файлів роби так: `UC-01_Order_Process.bpmn`, `BPMN_Monthly_Report.svg`, `ERD_v1.png`.
